"WP E-Commerce Extra Shipping Options" module gives the clients the ability to set the various new postage options.

== Description ==

We created “WP E-Commerce Extra Shipping Options” as a FREE wordpress module to give the clients the ability to set the following postage options.

* UK - 1st Class:
* UK - Recorded Delivery:
* UK - Special Delivery (next day):
* UK - Saturday Guaranteed Special Delivery:
* Europe - International Signed For:
* Rest of the World - International Signed For:

If the price of a shipping method is set to £0.00 the option will not be displayed to the end user.

These can be customised to suit other countries on request.

More information can be found on the plugins website <a href=\"http://www.dijituldevelopments.co.uk/wordpress-plugins/wp-e-commerce-plugins/\">WP E-commerce plugins</a>

== Installation ==

1. Upload `wp-e-commerce-shipping-module` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the postage settings of WP E-commerce
